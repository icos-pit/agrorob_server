## Installation

```
$ python -m pip install Django
```
## Running the application
Open a new terminal
```
$ source /opt/ros/<ros_distro>
$ ros2 bag play <bag file> --loop
```
Open a new terminal
```
$ ./zenoh-bridge-dds -m peer
```
Open a new terminal
```
$ cd agrorob_server
$ python manage.py runserver 
```
